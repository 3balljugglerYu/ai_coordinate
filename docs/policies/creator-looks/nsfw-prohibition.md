# NSFW 全面禁止 (P-6 ドラフト)

> **ステータス**: Stage 3 公開前ドラフト / 法務レビュー前

## 1. 禁止対象 (= MVP では全面禁止)

Creator Looks に **以下を含む画像** は投稿禁止:

- **性的表現**: 露骨な性的描写 / 過度な肌露出 / 性的暗示の強い描写
- **暴力表現**: 流血 / 殺傷 / 拷問 / 自傷の描写
- **差別表現**: 人種・性別・宗教・性的指向・障害等を侮蔑する表現
- **未成年への性的な描写** (= 国際法上の最重大違反として即時通報)
- **薬物・違法行為の助長**

## 2. 投稿時の自己申告

投稿フォームの同意項目 5「センシティブな要素を含まない」に
チェックすることで、投稿者は上記禁止対象に該当しない旨を **誓約** する。

虚偽申告は本サービスへの重大な信頼違反として、アカウント凍結 / 削除の対象となる。

## 3. 運営の確認

投稿された全画像は運営が人手で確認する (= 計画書 ADR、MVP 期は自動判定なし)。
NSFW 該当が疑われる投稿は **即時非公開化** し、運営内で精査する。

## 4. 通報

(Stage 3 で本格実装予定) 公開後に NSFW 要素を発見した場合、
消費者は投稿詳細ページから通報可能。通報があれば運営が再審査する。

## 5. 違反時の対応

- 軽微 (= 境界事例): 投稿の非公開化 + 注意通知
- 中程度 (= 規約への抵触): 投稿の非公開化 + 24〜72 時間の投稿停止
- 重大 (= 未成年への性的内容など): アカウント即時凍結 + 関係当局への通報

## 6. 将来の方針

MVP では全面禁止だが、Stage 3 以降に **「センシティブ」ゾーン** を設けて
年齢制限ゲート付きで一部のコンテンツを許容する可能性を検討する
(= 別途規約改定 + UI 改修が必要)。

## 7. 改定履歴

- v1.0 (ドラフト): 初版作成 (= MVP 全面禁止)
