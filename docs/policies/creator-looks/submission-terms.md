# Creator Looks 投稿規約 (P-1 ドラフト)

> **ステータス**: Stage 3 公開前ドラフト / 法務レビュー前
> 公開時期: Persta.AI で Creator Looks 機能を全公開する直前

## 1. 適用範囲

本規約は Persta.AI (以下「本サービス」) の **Creator Looks** 機能において、
ユーザー (以下「クリエイター」) が投稿する画像コンテンツに適用される。
本サービス全体の利用規約・プライバシーポリシーと併せて適用される。

## 2. 投稿可能な画像

Creator Looks に投稿できるのは、以下のいずれかに該当する画像に限る。

1. クリエイター本人が制作した画像 (= イラスト / グラフィック / 撮影)
2. 他者の作品で、**明示的かつ書面・電子記録による許諾** を得たもの

下記の画像は投稿禁止:

- 第三者の知的財産 (アニメ・ゲーム・ブランドロゴ・キャラクター等) を含むもの
- 性的・暴力的・差別的表現を含むもの (= NSFW)
- 個人を特定可能な写真・自画像で、被写体本人の同意を得ていないもの
- 違法行為を助長するもの

## 3. 投稿時の同意事項

投稿時に以下 5 項目すべてに同意する必要がある (= 投稿フォームのチェックボックスで明示)。

1. この画像の **著作権を侵害していない**
2. 第三者の **知的財産 (アニメ・ゲーム・ブランド等) を侵害していない**
3. この画像の **衣装と背景が他ユーザーの着せ替え素材として使われる** ことに同意する
4. Persta.AI が **サンプル・宣伝・ヘルプ等でこの画像を使う** ことに同意する
5. この画像は **センシティブな要素を含まない**

同意した時点で記録されるタイムスタンプは `user_style_templates.submission_consents` に
JSONB で保存される。

## 4. 利用範囲

投稿された画像は以下の目的で利用される。

- 本サービス内で他ユーザーが「自分のキャラに衣装を着せる」素材として
- 本サービスのホーム画面カルーセル / 詳細ページ / マーケティング素材で
- 本サービスの ヘルプ / FAQ / ニュースリリース / SNS シェア素材で

利用時にはクリエイターの帰属表示 (= Persta ニックネーム + プロフィールリンク) を
**原則として併記** する (詳細は `attribution-policy.md` 参照)。

## 5. 内部処理

投稿画像は本サービスの内部処理を通じて以下のように扱われる。

- 投稿時に **EXIF / 位置情報 / 撮影メタデータを除去** し、WebP 形式に再エンコード
- 別途、画像から **衣装・背景・小物を抽出した内部プロンプト** を AI で生成し、
  本サービス内部にのみ保管 (= クリエイター本人含む第三者には公開しない)
- 内部プロンプトの生成には OpenAI 社の Vision Language Model を利用するため、
  当該画像は処理時に米国の OpenAI サーバへ送信される (= 第三国移転、別途プライバシー
  ポリシーに従う)

## 6. 投稿頻度制限

クリエイター 1 人あたり 24 時間に **2 件まで** 投稿可能。
スパム / 低品質な大量投稿を防ぐためのルールであり、運用状況に応じて変更する場合がある。

## 7. 撤回 / 削除

クリエイターはいつでも自身の投稿を撤回 (= 非公開化) または削除できる。
詳細は `withdrawal-policy.md` 参照。

## 8. 違反時の対応

本規約に違反した投稿が確認された場合、本サービスは以下の措置を取る:

- 投稿の即時非公開化
- クリエイターへの警告通知
- 重大・反復違反の場合はアカウント凍結 / 削除
- 必要に応じて関係当局・権利者への通報

## 9. 規約改定

本サービスは本規約をいつでも改定できる。重要な改定はクリエイターに通知し、
継続利用をもって新規約への同意とみなす。
