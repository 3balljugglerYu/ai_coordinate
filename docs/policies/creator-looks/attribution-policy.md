# 帰属表示ポリシー (P-5 ドラフト)

> **ステータス**: Stage 3 公開前ドラフト / 法務レビュー前

## 1. 帰属表示の場所

Creator Looks 投稿に対するクリエイター帰属は以下の場所で表示される:

| 場所 | 表示内容 | リンク先 |
|---|---|---|
| ホームカルーセル | 「Creator Looks」バッジ | 投稿詳細ページ |
| 投稿詳細ページ | `by {Persta ニックネーム}` | `/users/[userId]` プロフィールページ |
| 生成結果ギャラリー | `by {ニックネーム}` (Phase 7 以降) | 同上 |
| X 共有テキスト | `by {ニックネーム}` (= プレーンテキスト、リンクなし) | OGP 経由で本サービスへ |

## 2. ニックネーム表示

帰属表示には **本サービスの Persta ニックネーム** を使用する。
クリエイターはマイページ → プロフィール編集からニックネームを変更可能。
ニックネーム未設定の場合は「匿名クリエイター」と表示される。

**X アカウント名は使用しない** (= プライバシー配慮、本人意思の尊重)。

## 3. ニックネーム変更の影響

クリエイターがニックネームを変更すると、以下のすべてに新しいニックネームが反映される:

- ホームカルーセルの帰属表示
- 投稿詳細ページの「by」表示
- 通知メール本文 (= 過去通知も含めて、最新ニックネームで表示される)

過去の X 共有テキストに含まれた旧ニックネームは変更されない (= 既に拡散済みのため
技術的に回収不能)。

## 4. 帰属表示なしでの利用

本サービスは Creator Looks 投稿を **帰属表示なしで利用することはない**。
仮にマーケティング素材として転載する場合も帰属を併記する。
帰属なしの利用希望はクリエイターご本人とは別途協議の上、書面で合意する。

## 5. 「匿名」希望

クリエイターが本サービス内でも完全匿名を希望する場合:

- ニックネームを「匿名クリエイター」のような汎用語に変更する
- 投稿には常に「匿名クリエイター」と表示される
- プロフィールページからの本人特定は技術的に困難になる (= URL の userId は UUID)
