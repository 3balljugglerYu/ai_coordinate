# クリエイター行動規範 (P-2 ドラフト)

> **ステータス**: Stage 3 公開前ドラフト / 法務レビュー前

## 1. 趣旨

Creator Looks は「クリエイターの作品が他のユーザーの着せ替え素材として使われる」
仕組みのため、クリエイター・消費者・運営のすべての行動が他者の創作物 / 個人情報 /
プライバシーに影響する。本規範はクリエイターに期待される最低限の行動基準を示す。

## 2. 守るべきこと

- **他者作品の無断利用をしない**: 自作 / 撮影 / 明示許諾済みのみ投稿
- **出所申告は正確に**: フォーム「画像の出所」(self_created / self_photographed /
  licensed_other) は虚偽申告しない
- **第三者の IP を侵害しない**: アニメ / ゲーム / ブランド / 著名キャラの模倣は避ける
- **個人特定情報を含めない**: 他人の顔写真・電話番号・住所等の特定情報を画像に
  含めない (= 自分自身の写真は OK)
- **センシティブ表現は投稿しない**: 性的・暴力的・差別的表現は MVP では全面禁止

## 3. 期待される姿勢

- **建設的な投稿**: 着せ替えの楽しさを広げる作品の投稿
- **他クリエイターを尊重**: 他クリエイターの作品の盗用 / 不当な批判は禁止
- **運営との対話**: 通知に返信できないシステム仕様だが、要望はマイページ
  サポート問い合わせから

## 4. 違反時の対応

- 軽微違反: 投稿の非公開化 + 注意通知
- 中程度違反: アカウントの一時的な投稿停止 (= 24〜72 時間)
- 重大違反: アカウント凍結 / 削除、必要に応じて関係当局・権利者へ通報

## 5. 通報

他クリエイターの違反を発見した場合は、本サービスの通報窓口
(= Stage 3 で実装予定) から運営にご連絡ください。
