# DMCA / 著作権侵害申し立てポリシー (P-7 ドラフト)

> **ステータス**: Stage 3 公開前ドラフト / 法務レビュー前

## 1. 申し立て窓口

著作権侵害が疑われる Creator Looks 投稿を発見された場合、
以下のいずれかの方法で本サービスにご連絡ください。

- **専用メール**: copyright@persta.ai (= Stage 3 公開前に設定)
- **問い合わせフォーム**: https://persta.ai/contact (= 件名「Creator Looks 著作権申し立て」)

## 2. 申し立てに必要な情報

申し立て時には以下の情報を必ずご提供ください。

1. 権利者または権利者代理人の氏名 / 連絡先 / 物理住所
2. 侵害されたとされる **元の著作物の特定情報** (= URL / 作品タイトル / 作者名)
3. 本サービス内で侵害されている **投稿の URL** (= https://persta.ai/inspire/[templateId])
4. 申し立て内容の **誠実な信念** に基づく宣誓 (= 虚偽申し立ては偽証罪に該当する旨理解)
5. 申し立て内容が **正確** である宣誓 (= 権利者または許可を得た代理人として行動)
6. **物理的または電子的な署名**

(= 米国 DMCA セーフハーバー (17 U.S.C. § 512(c)(3)) に準拠した形式)

## 3. 処理 SLA

- **24 時間以内**: 該当投稿の予防的非公開化 (= `moderation_status='removed'`)
- **48 時間以内**: 申し立て内容の精査 + 投稿者への通知
- **7 営業日以内**: 最終判定 (= 永久削除 / 復活 / アカウント措置)

## 4. 反対通知 (Counter-Notice)

投稿者は申し立てに対し **反対通知** を提出できる。
反対通知は本サービスから申し立て者に転送され、
申し立て者が訴訟提起しない限り **10〜14 日後に投稿は復活** する可能性がある
(= DMCA § 512(g))。

反対通知に必要な情報は申し立てと同様の形式で受け付ける。

## 5. 反復違反者ポリシー

複数回 (= 概ね 3 回以上、運営判断) の有効な著作権侵害申し立てを受けたクリエイターは、
アカウントを永久凍結する。

## 6. 虚偽申し立てへの対応

虚偽の DMCA 申し立てを行った場合、米国法上は偽証罪 / 損害賠償の対象となる
(17 U.S.C. § 512(f))。本サービスとしても虚偽申し立て者への対応 (= 反対通知サポート等)
を行う。

## 7. 本サービスの責任範囲

本サービスはユーザー投稿コンテンツのプラットフォームであり、
個別投稿の著作権侵害責任は **第一義的にクリエイター本人** にある。
本サービスは申し立てに迅速・誠実に対応することで DMCA セーフハーバー条項の
保護を受ける。
